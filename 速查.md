# 速查

## adb

### 启动

 adb shell am start -n package/activity
 打开App，在cmd窗口中输入以下命令。
 adb shell dumpsys window windows | findstr “Current”
 /之前的就是package，之后的就是当前的activity

### 点击

 .\adb shell input tap x y
 滑动
 adb shell input swipe startX startY endX endY 500（ms）
